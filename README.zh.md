# qtpyeditor

#### 介绍
一个基于qtpy的代码编辑器，由纯Python代码编写，兼容PySide2和PyQt5。

这些代码可以用来解决PySide2下面缺少QScintilla绑定，而且web编辑器较为费资源的问题

该编辑器支持行号渲染、代码渲染、智能提示，但目前只有简单的例子。
